# Akili Za Kesho | Week 2 Deployment Portal

This repository hosts the interactive deployment portal for the Week 02 internship program. 

## Features
- Cinematic typewriter gateway sequence.
- Interactive terminal orchestration simulation.
- Responsive dashboard interface.
- Deployed via Vercel for high-availability edge delivery.

## Setup Instructions
1. Clone the repository.
2. Push to a personal GitHub repository and link to Vercel/Netlify.
